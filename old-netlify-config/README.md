# Архив настроек Netlify

Эта папка содержит оригинальные файлы конфигурации для развертывания на Netlify.

## Содержимое

- `netlify.toml` - Конфигурационный файл Netlify
- `NETLIFY_DEPLOYMENT.md` - Полное руководство по развертыванию на Netlify

## Использование

Если вы хотите вернуться к развертыванию на Netlify:

1. Скопируйте `netlify.toml` в корень проекта
2. Следуйте инструкциям в `NETLIFY_DEPLOYMENT.md`

## Текущая платформа

Проект настроен для развертывания на **Cloudflare Pages**.

См. `CLOUDFLARE_DEPLOYMENT.md` в корне проекта для инструкций.
